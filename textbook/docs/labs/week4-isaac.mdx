# Lab 4: Isaac Sim Navigation

## Objective
Use Isaac Sim to navigate a warehouse environment.

## Steps

1.  **Launch Isaac Sim**:
    Open the "Warehouse" environment.
2.  **Add Carter Robot**:
    Add the NVIDIA Carter robot to the scene.
3.  **Enable ROS 2 Bridge**:
    Enable the ROS 2 Bridge extension.
4.  **Teleoperate**:
    Use `ros2 run teleop_twist_keyboard teleop_twist_keyboard` to drive the robot.

## Deliverable
A short video recording of the robot navigating the warehouse.
