# Vercel

Vercel is the creators of Next.js and provides the best deployment experience.

## Deploying to Vercel

1.  Push your code to a Git repository (GitHub, GitLab, Bitbucket).
2.  Import your project into Vercel.
3.  Vercel will automatically detect Next.js and configure your build settings.
4.  Your application is deployed!

## Features

Vercel provides features like:

-   **Zero Configuration**: Automatic scaling, global CDN, and edge caching.
-   **Preview Deployments**: Every pull request gets its own live preview URL.
-   **Analytics**: Real-time insights into your application's performance.
-   **Edge Functions**: Run code closer to your users.

## Environment Variables

You can configure environment variables in the Vercel dashboard or using the Vercel CLI.

```bash
vercel env add MY_VARIABLE
```
